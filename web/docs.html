General Commands

;invite → Displays the join guild link of the bot.
;ping → Runs a connection test to Discord.
;help → Display help for a command.
;stats → Gets information on the bot.
;uptime → Gets uptime info on the bot.

Moderation Commands

;ban → Bans a user.

;kick → Kicks a user.

;purge → Prunes a certain amount of messages w/o filter.
;unban → Unbans a member
